Multipaz Identity Reader Backend

The Multipaz Identity Reader application uses this backend for certification of hardware-backed single-use keys used for reader authentication according to ISO/IEC 18013-5:2021. These keys are signed by the following root certificate: Reader Root Certificate

In case the Multipaz Identity Reader application is running on a device where it wasn't possibly to establish a high-level of trust (for example if Verified Boot is disabled) or if the application wasn't signed with the key used to sign APKs distributed on https://apps.multipaz.org (for example if you built it yourself from source), the following root certificate is used instead: Reader Root Certificate for Untrusted Devices

This reader backend is part of the Multipaz project and is considered experimental software. Use at your own risk.
